# Delhi Metro Route Finder

A Python desktop application built using Tkinter that helps users find the shortest route between Delhi Metro stations.

## Features

* Find the shortest metro route between stations
* Graph-based station network representation
* Breadth-First Search (BFS) route calculation
* User-friendly GUI built with Tkinter
* Interactive source and destination selection

## Technologies Used

* Python
* Tkinter
* Graph Data Structure
* Breadth-First Search (BFS)

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Delhi_Metro.git
```

2. Navigate to the project folder

```bash
cd Delhi_Metro-main
```

3. Run the application

```bash
python dmrc.py
```
